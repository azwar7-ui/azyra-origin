## 🚀 Overview
Azyra Origin is a lightweight, secure chat platform built for users who value privacy and digital scarcity. No clutter, no invasive tracking—just secure, ID-based interactions.

## 🛠️ Engineering Foundation
The project features a robust, modern backend architecture designed for scale and security.

### 🛡️ Security & Performance
- **Salted SHA-256 Hashing:** Advanced password protection using unique user-specific salts to negate rainbow table attacks.
- **Edge Computing:** All authentication and message handling is routed through Supabase Edge Functions, keeping the database isolated and secure.
- **HMAC Session Management:** Signed session tokens with a 30-day expiry for seamless, secure persistence.
- **Traffic Control:** Integrated DB-level rate limiting (20 msgs/30s) to prevent spam and flooding.
- **Optimistic UI:** Real-time message rendering for zero-latency user experience.

## 🏆 The Azyra Hierarchy (Verification)

### 🔱 The Golden Tick (Available Now)
The **Golden Tick** is an exclusive, non-recurring legacy badge. It is hard-coded to be awarded **only to the first 500 users** who join the Azyra ecosystem. 
- Once the 501st user signs up, the Golden Tick is retired forever.
- It represents the "Founding Class" of Azyra Origin.

### 💎 Future Verification Tiers
As the platform scales (targeting 5k+ users), we will introduce specialized verification tiers for distinguished users and contributors:
- **Obsidian / Amethyst** (Elite Tiers)
- **Diamond / Platinum / Silver** (Subscription-based & Verified Status)
