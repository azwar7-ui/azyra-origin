# 🌌 Azyra Origin

A lightweight, browser-based real-time private chat system 
built on custom identity — no phone number, no email. 
Just your Azyra ID.

---

## 🚀 Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES Modules)
- **Backend:** Supabase (PostgreSQL + Realtime)
- **Hosting:** Netlify
- **Auth:** Custom (SHA-256 hashed passwords + Azyra ID system)

---

## ✨ Features

- 🔐 Custom auth — signup/login with username + password
- 🪪 Unique Azyra ID generated per user (format: az_xxxxxx)
- 👥 Add friends by Azyra ID
- 💬 Real-time 1-to-1 private chat
- 🌙 Dark / Light theme toggle
- 📱 Mobile responsive

---

## 🛠️ Setup

No install needed. It's a single HTML file.

1. Clone the repo:
   git clone https://github.com/azwar7-ui/azyra_origin.git

2. Open index.html in your browser
   OR deploy directly to Netlify
